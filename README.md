# CareerFlow

CareerFlow is a portfolio-ready career management app for tracking job applications, interviews, offers, deadlines, and notes.

## Tech Stack

- React
- TypeScript
- Express
- MySQL
- SASS

## Project Structure

- `client` - React frontend built with Vite
- `server` - Express API written in TypeScript

## Getting Started

1. Install dependencies from the project root:
   `npm install`
2. Start the API:
   `npm run dev:server`
3. Start the frontend in a second terminal:
   `npm run dev:client`

Frontend URL: `http://localhost:5173`

API health check: `http://localhost:4000/api/health`

## Environment Files

- `client/.env.example`
- `server/.env.example`
